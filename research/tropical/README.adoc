== What This Is

Tropical semirings (max-plus and min-plus) provide a compositional algebra for
tracking resource bounds and access costs in type systems.  Two complementary
uses in the pipeline:

* **Max-plus (throughput)** — used in session-type protocol resource typing.
  Formalised in Isabelle in `hyperpolymath/tropical-resource-typing`.
  The star `A*` computes the maximum-weight simple-path matrix: under
  `no_pos_cycle`, `A*(i,j)` = the heaviest walk from `i` to `j`.

* **Min-plus (latency)** — used in typed-wasm L11 cost-tracking.
  `A*(i,j)` = the cheapest access path from field `i` to field `j`.
  Same algebra, dual order.

Both are instances of the same abstract closed semiring.  The Isabelle proofs
cover max-plus; by duality they imply min-plus.

== Open Questions

. The star equation requires `no_pos_cycle` in the Isabelle proof.  What is
  the Idris2 equivalent?  A type-level constraint on the matrix?  A proof
  that the access graph is acyclic (for memory — it always is)?

. In typed-wasm, the access graph between fields *is* always acyclic (no field
  contains itself).  So `no_pos_cycle` is structurally guaranteed.  Can this
  be encoded at the type level so the star equation holds unconditionally for
  typed-wasm access matrices?

. The CNO characterisation (`is_trop_cno`) relates to session-type normal forms.
  Can it encode the "this session has been reduced to normal form" invariant in
  a QTT context?
